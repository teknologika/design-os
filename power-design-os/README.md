# Design OS Power

This power packages Design OS workflows, guidance, and MCP tools for Kiro.

## Install (Local Path)
1. Open Kiro → Powers panel → Add power from Local Path
2. Select this folder: `power-design-os/`
3. Test activation by mentioning keywords like “product roadmap” or “design tokens”

## Install (GitHub)
Add power from GitHub using:
- https://github.com/teknologika/design-os-mcp

## What’s Included
- `POWER.md` with onboarding, steering map, MCP servers, and rules
- `mcp.json` with MCP server commands
- `steering/` docs for each workflow stage

## Notes
- This power documents Design OS slash commands but does not add new slash commands to Kiro.
- Use shadcn/ui in the target codebase when implementing exports.
